# my dots

my hyprland setup on arch. nothing fancy, just how i like things.

## fresh install

- run the text files in this order: `core-pkgs.txt`, `nvidia-install.txt` (if nvidia), `post-install.txt`
- copy the `.config` stuff where it belongs and make `monitors.conf` and `nvidia.conf` from the examples in `hypr/host/`
- reboot and `first-boot.sh` does the rest

## what's in here

- **hyprland** — window manager config split into `base/` (shared) and `host/` (per-machine, not tracked)
- **waybar** — top bar
- **wofi** — app launcher
- **kitty** — terminal
- **mako** — notifications
- **fastfetch** — the thing that shows system info when i open a terminal
- **gtk 3 & 4** — dark theme stuff
- **wallpapers** — in `images/wallpapers/`
- **bashrc** — shell stuff
